Test and Configure Your System         

This page will help you test and configure your system so that you can access this course. Below, in the "Short Version", is a summarized list of the applications/plugins that you need to have installed to be able to take advantage of all of the content throughout the course. If you need more information, or are not sure if you have the right technologies installed, there are tests provided further down in the "Long Version".

Short Version:
Here is a list of the applications and plugins that you will need:
  • A supported browser:
    • Windows: IE 5.5 and 6.0, Netscape 7.0
    • Mac OS: IE 5.2 for all content except Minitab files and StatTutor (more info)
  • Flash Player 6 (more info)
  • Minitab: CMU students can download Minitab here. Note that Minitab is only available for the Windows platform (more info)
  • Javascript enabled (more info)
  • Java installed and enabled (more info)
  • Pop-up windows enabled (more info)
  • A sound card and speakers/headphones (more info)
Long Version:

Below is additional information about the applications/plugins that are required for this course, along with some simple tests that can check to see if your system is compatible with the course requirements. Some links will go to another page using this browser window. If there is no 'Back" button on the page, just use the browser Back button to return here. Some links may "pop up" a new browser window, just use the X button in the upper right corner to close the window.

Supported Browsers:
  • Windows: IE 5.5 and 6.0, Netscape 7.0
  • Mac OS: IE 5.2 for all content except Minitab files and StatTutor. Minitab files cannot be viewed on the Mac because there is no longer a version of Minitab made for the Mac. Although StatTutor will eventually be supported for the Mac, it currently has problems that prohibit it from running correctly. However, you can still use a Mac to access the bulk of the course content, and then use a cluster pc to do the exercises that require Minitab. Note that the course does not currently work in Safari.
  • Since this course is still fairly new, we have not had the chance to ensure that the content works correctly in other browsers or on other platforms. Eventually the list of supported browsers and platforms will be expanded.
 MacroMedia Flash Player
  • Macromedia Flash Player is used throughout the course. Version 6 or higher of the Flash Player is required.
  • Flash Test - This test will load a new page with a small flash applet that will tell you what version of Flash you are using. Click here to test.
  • If you need to update your version of Flash, you can do that here
Minitab
  • Minitab is also used throughout the course, mostly in the "Learn By Doing" activities. CMU students can download Minitab here. Minitab is now only available for the Windows platform. Mac users are still able to access the bulk of the course content, but will have to use a cluster pc for some of the "Learn By Doing" activities and for "StatTutor", which also requires Minitab.
Javascript
  • JavaScript is used in this course for navigation, launching learning activities, and by StatTutor. You must enable it in your web browser before using the course.
  • To enable javascript in Windows:
    • In Internet Explorer, click on Tools -> Internet Options -> Security -> Custom Level -> category Scripting, item Active Scripting, select "Enable".
    • In Netscape, click Edit -> Preferences -> category Advanced, item Scripts & Plugins, select "Enable JavaScript for Navigator".
  • To enable javascript on the Mac:
    • In Internet Explorer, click on Explorer -> Preferences -> category Web Content, item Active Content, select "Enable scripting"
 Java
  • Some of the learning activities in this course feature interactive java applets. To be able to use these, you must have java installed and enabled.
  • Java Test - This test will load a new page with a small java applet. If you see a grey box with an X, or nothing at all, you should use the instructions below to make sure that you have java enabled, and then try again. If you still can't see the applet, use the link below to update your version of java. Click here to test.
  • To enable java in Windows:
    • In Internet Explorer, click on Tools -> Internet Options -> Advanced -> category Java, select "Use Java...".
    • In Netscape, click Edit -> Preferences -> category Advanced, select "Enable Java".
  • To enable java on the Mac:
    • In Internet Explorer, click on Explorer -> Preferences -> category Java, item Java Options, select "Enable Java"
  • If you need to update your version of java, you can do that here
Pop-Up Windows (Advertisement Blocking)
  • If you have configured your web browser to not allow pop-up windows or if you have installed any utility program claiming to block "pop-up ads" then you will not be able to access much of the content in this course.
  • Pop-Up Test - This should produce a second small browser window. Click here to test. If you do not get a second window, your browser may need to be configured, or you may have ad-blocking software installed.
  • If you are using Netscape 7, you should make sure that your browser is configured as follows: Click Edit -> Preferences -> category Advanced, make sure that "Open a link in a new window" is checked
Sound
  • There are short movies throughout this course that include audio. To be able to hear the audio, your computer must have a sound card and speakers or headphones attached.
