<p>To start, we'll create a table that will list each statistical measure and the value for each college. When you get done, your table should look like this:</p><p><img src="../../../webcontent/excel_instructions/boxplot3lbd1_old_1.gif"/></p><ul><li>To create this table, first make the row and column labels: in an empty section of your worksheet, pick a cell and type <code style="inline">Statistic</code>. Then move one cell to the right and type <code style="inline">College A</code>. Move one cell to the right and type <code style="inline">College B</code>. Keep doing this for colleges C, D, E, and F. Now, in the cell underneath <code style="inline">Statistic</code>, type <code style="inline">Q1</code>. Move down one cell and type <code style="inline">min</code>, then underneath that <code style="inline">Median</code>, then <code style="inline">Max</code>, and finally <code style="inline">Q3</code>.</li><li>Note that the order here is slightly different than the ordering we've been using - the reason is that the statistical measures have to be in this order (Q1, min, Median, Max, Q3) for Excel to graph the boxplot correctly.</li></ul>
